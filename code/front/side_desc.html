🧭 Sidebar.jsx 상세 코드 리뷰

1. 레이아웃 제어 : sidebarWidth

const sidebarWidth = isMobile ? '100%' : (isCollapsed ? '80px' : '240px');
[분석] 모바일에서는 화면을 꽉 채우고, 데스크톱에서는 펼침/접힘에 따라 너비를 동적으로 변경합니다.
[핵심] transition: 'width 0.35s ...'를 통해 접히거나 펼쳐질 때의 부드러운 애니메이션을 구현했습니다.

2. 🔼 SVG 기반 토글 버튼

<svg style={{ transform: isCollapsed ? 'rotate(0deg)' : 'rotate(180deg)' }} ...>
[분석] 화살표 아이콘의 회전 각도를 상태값에 따라 0도나 180도로 전환합니다.
[의도] 사용자에게 현재 상태가 '확장 모드'인지 '축소 모드'인지 직관적으로 알려줍니다.

3. 메뉴 활성화 상태 관리

backgroundColor: activeTab === menu.id ? 'rgba(99, 102, 241, 0.1)' : 'transparent',
borderLeft: (!isMobile && !isCollapsed && activeTab === menu.id) ? '4px solid #6366f1' : 'none',
[분석] activeTab과 메뉴의 `id`를 비교해 선택된 메뉴에 강조 스타일(배경색 및 좌측 테두리)을 적용합니다.
[의도] 관제 화면이 복잡할 때 현재 어느 탭을 보고 있는지 즉시 인지하도록 돕습니다.
