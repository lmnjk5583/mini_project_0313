📺 VideoPanel.jsx 상세 코드 리뷰

1. SingleMedia : 스트리밍 엔진

if (Hls.isSupported()) {
  const hls = new Hls();
  hls.loadSource(url);
  hls.attachMedia(videoRef.current);
}
[라인 분석] hls.js 라이브러리를 사용하여 브라우저 기본 기능이 HLS를 지원하지 않아도 고화질 영상을 끊김 없이 재생합니다.

2. VideoPanel : AI 분석 모드 전환

if (isReverseAnalysis) {
  finalUrl = `.../its/video_feed?url=${...}`;
  isHls = false;
}
[핵심] 화면 인덱스(idx)에 따라 일반 CCTV 영상인지, 서버에서 분석 중인 'AI 피드'인지 결정합니다.
[의도] AI 분석 데이터는 실시간성이 중요하므로 MJPEG 스트림(`isHls = false`)으로 즉각 송출합니다.

3. handleCapture() : 즉시 기록 시스템

const res = await axios.post(`.../api/capture_now`, { ... });
const { value: text } = await Swal.fire({ ... }); // 캡처 후 메모 입력
[핵심] 사용자가 캡처 버튼을 누르면 서버에서 *현재 프레임*을 즉시 따낸 뒤, 추후 메모를 업데이트하는 비동기적 방식을 사용합니다.

4. 모달 확대 (Expanded View)

{expandedMedia && ( <div style={modalOverlayStyle} onClick={...}> ... )}
[핵심] CCTV 4분할 화면에서 특정 영상을 클릭하면 expandedMedia 상태가 변경되어 모달이 활성화됩니다.
[의도] 좁은 관제 화면에서 사고 발생 시, 특정 채널을 전체 화면으로 확대해 정밀 관제를 하기 위함입니다.
