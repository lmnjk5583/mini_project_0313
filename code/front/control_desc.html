⚙️ ControlPanel.jsx 상세 코드 리뷰

1. 시뮬레이션 제어부 (상단 버튼)

<button onClick={() => startSim("fire")} ... >🔥 화재 상황 공유</button>
<button onClick={() => startSim("reverse")} ... >🚗 역주행 상황 공유</button>
[라인 분석] startSim 함수를 호출하여 서버에 시뮬레이션 트리거를 전달합니다.
[의도] 실제 사고가 발생하지 않아도 시스템이 잘 작동하는지 테스트하거나, 관제사 교육용 상황을 강제로 발생시킵니다.

2. 실시간 시스템 로그 (하단 콘솔)

<div style={{ overflowY: 'auto', ... }}>
  {logs.map((log, i) => (
    <div key={i} style={{ color: log.includes('🚨') ? '#f87171' : '#4ade80' }}>{log}</div>
  ))}
</div>
[라인 분석] logs 배열을 매핑하여 시스템의 모든 이벤트를 역순으로 보여줍니다.
[핵심] log.includes('🚨') 조건을 사용해 긴급 상황(🚨)과 정상 동작(✅ 등)을 색상으로 구분합니다. 직관적인 시각적 피드백을 제공하죠.

3. UI/UX: flex: 1의 마법

<div style={{ flex: 1, display: 'flex', flexDirection: 'column', minHeight: 0 }}>
[분석] 로그 영역에 flex: 1과 minHeight: 0을 설정했습니다.
[의도] 화면 크기가 변해도 로그 영역이 부모를 넘치지 않고, 내부에서만 스크롤이 발생하도록 설계된 '전문가형 레이아웃'입니다.
