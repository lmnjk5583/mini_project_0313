📊 StatsPanel.jsx 상세 코드 리뷰

1. fetchStats() : 실시간 통계 업데이트

const response = await axios.get(`.../api/stats/summary?mode=${dataMode}`);
// 5초마다 데이터 갱신 (Polling)
const interval = setInterval(fetchStats, 5000);
[라인 분석] dataMode(실제/시뮬/전체)를 쿼리 파라미터로 넘겨 서버에서 필터링된 통계값을 가져옵니다.
[의도] 대시보드를 보는 동안에도 통계 수치가 실시간으로 변하도록 5초 단위의 Polling 방식을 채택했습니다.

2. 이벤트 분포 시각화 (진행 바)

<div style={{ ...barProgressStyle, width: `${calculatePercent(count)}%` }} />
[라인 분석] CSS-in-JS를 활용하여 width 값을 백분율로 계산해 바(Bar)의 길이를 동적으로 조절합니다.
[핵심] 관리자가 현재 어떤 유형의 사고가 가장 많은지 시각적으로 즉시 파악하게 해주는 핵심 UI 요소입니다.

3. historyLogs : 상세 데이터 감사

const response = await axios.get(`.../history?date=${selectedDate}&mode=${dataMode}`);
[라인 분석] 특정 날짜(selectedDate)와 모드에 맞는 상세 로그 리스트를 가져옵니다.
[의도] 과거 사고 이력을 추적하여 나중에 정탐/오탐의 근거를 확인하는 '감사 기록(Audit Trail)' 기능을 수행합니다.
